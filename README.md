# HFTA (High-Frequency Trading Analysis)

这是一个基于Streamlit开发的高频交易分析仪表板，用于展示和分析黑色金属冶炼行业的经济指标。

## 功能特点

- 实时数据展示
- 交互式图表
- 扩散指数分析
- 多维度数据对比

## 在线访问

访问地址：[Streamlit Cloud](https://hfta.streamlit.app)

## 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/BichengNiu/HFTA.git
cd HFTA
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行应用：
```bash
streamlit run dashboard_fixed.py
```

## 数据来源

- 黑色金属冶炼行业数据
- 经济指标数据
